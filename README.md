# workout-sns-api
Spring Boot REST API for Workout SNS

## 기술 스택

- Java 17
- Spring Boot 4.0.x
- MySQL 8.0
- Redis 7.2
- Docker
- JWT Authentication

## 로컬 실행

### 1. Docker로 DB 실행
```bash
docker-compose up -d
```

### 2. Spring Boot 실행
```bash
./gradlew bootRun
```
